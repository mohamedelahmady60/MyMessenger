## Messenger Real Time Chat App
this is a mobile messaging app used for instant messaging,  sharing photos, videos, and locations.

## Features

- Facebook Log In
- Google Sign In
- Email/Pass Registration / Log In
- Photo Messages
- Video Messages
- Real Time Conversations
- Location Messages
- Search for Users
- Deleting Conversations
- User Profile
- Dark Mode Support
